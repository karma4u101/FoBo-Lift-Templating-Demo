BootstrapScriptHelper

BootstrapScriptHelper is a helper object that implements some commonly used bootstrap component init/usage script functions. This convenience script functions is used in the FoBo.Bootstrap snippet but can also be used as a convenience helper in your own customized snippets.

    def someFunc = {

      :
    }
