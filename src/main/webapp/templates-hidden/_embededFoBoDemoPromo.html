Setup and Usage

For now you have to clone and build the module yourself but thats easy and straight forward so, if you have not done it already, go ahead and do it now, you will find all information you need to prepare the module for usage on Github FoBo module, then come back here for integration and usage.

To initiate and release the powers of the FoBo module chose the Toolkit option(s) you prefer and then call FoBo.init() from your lift projects Boot.

Using FoBo in your own project you should set up the init param for the toolkit module you need or if you are using the module defaults just call FoBo.init(). This demo uses the following unusually heavy module setup in lift Boot:


                           import net.liftmodules.FoBo

                           /**
                            * A class that's instantiated early and run.  It allows the application
                            * to modify lift's environment
                            **/
                            class Boot extends Loggable {
                              def boot {

                                //If using defaults FoBo init params can be omitted
                                FoBo.InitParam.JQuery=FoBo.JQuery171  
                                FoBo.InitParam.ToolKit=FoBo.FoBo020 
                                FoBo.InitParam.ToolKit=FoBo.PrettifyJun2011
                                FoBo.InitParam.ToolKit=FoBo.JQueryMobile101
                                FoBo.InitParam.ToolKit=FoBo.DataTables190
                                FoBo.InitParam.ToolKit=FoBo.Knockout200
                                FoBo.init()
                                  :

In Lift's html5 template:s add (depending on your setup) something like this:

                        <!DOCTYPE html>
                        <html>
                          <head>
                             :
                             <!--Main css for the FoBo020 module option-->
                             <link href="/classpath/fobo/fobo.css" type="text/css" rel="stylesheet">
                             <!--app specific tweeks and overrides external to module goes here-->
                             <link href="[path to your projects app css]" type="text/css" rel="stylesheet">  
                             :                        
                          </head>
                          <!--Demo page setup for prettyprint and bootstrap scrollspy-->
                          <body data-spy="scroll" data-target="#fobo-navbar" data-offset="40" onload="prettyPrint()">
                             :
                             <!--In end of page (for fast pageload) add the scripts you need--> 
                             <script src="/classpath/fobo/prettify.js" type="text/javascript"></script>
                             <script src="/classpath/fobo/lang-scala.js" type="text/javascript"></script>
                             <script src="/classpath/fobo/bootstrap-scrollspy.js" type="text/javascript"></script>
                             :
                          </body>        
                        </html>

Note the uniform setup without any version references in the html templates, making way for a smooth(er) toolkit upgrade. For more information see the FoBo modules Github.

The FoBo module comes with some convinience functions (snippets, optional to use) that will generate some commonly used toolkit helper scripts for you. For instansce doing it the Lift - FoBo way you can simply do this using a FoBo:s snippet containing some bootstrap convinience functions: 

                        <!DOCTYPE html>
                        <html>
                          <head>
                               :
                          <script data-lift="FoBo.ScriptHelper.registerLoadEventFactory"></script>         
                          <script data-lift="FoBo.Bootstrap.activateDropdown?on=.dropdown-toggle"></script>              
                          </head>
                          <body">
                               :
	                      <a href="#" rel="popover" title="The Title" data-content="The content ..." id="theId">...</a>
                              <script data-lift="FoBo.Bootstrap.popover?id=#theId&options=placement:'left'"></script>
                              <script data-lift="FoBo.Bootstrap.popoverPreventDefault"></script>
                               : 
                          </body>        
                        </html>

.... instead of doing this (a common bootstrap component init use case) 

                        <!DOCTYPE html>
                        <html>
                          <head>
                             :
		                  <script type="text/javascript">                                 		
			                function addLoadEvent(func) {
				              var oldonload = window.onload;
				              if (typeof window.onload != 'function') {
					            window.onload = func;
				              } else {
					            window.onload = function() {
						          if (oldonload) {
							        oldonload();
						          }
						          func();
					            }
				              }
			                }
			                addLoadEvent(function() {
				              $('.dropdown-toggle').dropdown()
			                });	   			               
		                  </script>                                 
                          </head>
                          <body">
                             :
                              <!-- Set a boostrap popover element -->   
	                      <a href="#" rel="popover" title="The Title" data-content="The content ..." id="theId">...</a>
                              <!-- Initiate the popover and place it on the left -->
	                      <script>
                                $(function () { 
                                  $("#theId").popover({placement:'left'});
                                }); 
                              </script> 
                              <!-- Prevent the default action for click on the popover -->
                              <script> 
                                $(function(){
                                  $("a[rel=popover]")
                                    .popover()
                                    .click(function(e) {
                                      e.preventDefault()
                                  });
                                });
                              </script> 
                               : 
                          </body>        
                        </html>

At this writing the available snippet helpers are few but expect the numbers to grow. You are wellcome to contribute code and/or ideas. Take a look att the API documentation on how to invoke the helpers and description on what result it produces for example look att the FoBo.ScriptHelper.registerLoadEventFactory snippet function by open FoBo.ScriptHelper page and click on the registerLoadEventFactory function name for details.

DataTables setup


A simple setup

                        <!DOCTYPE html>
                        <html>
                          <head>
                               :
                            <!-- Use the smoothness theme -->  
                            <link rel="stylesheet" type='text/css' href="/classpath/fobo/themes/smoothness.css"/>
                            <!-- <link rel="stylesheet" type='text/css' href="/classpath/fobo/themes/ui-lightness.css"/> -->
                            <link rel="stylesheet" type='text/css' href="/classpath/fobo/jquery.dataTables_themeroller.css"/>
                               :
                            <script type="text/javascript" charset="utf-8" src="/classpath/fobo/jquery.dataTables.js"></script>
                            <!-- Use the FoBo API to init the table (optional) -->
                            <script data-lift="FoBo.DataTables.dataTable?id=#example&options='bJQueryUI':true,'sPaginationType':'full_numbers'"></script>
                          </head>
                          <body">
                               :
                            <table cellpadding="0" cellspacing="0" border="0" class="display" id="example">
                               :
                            </table>   
                          </body>        
                        </html>

The FoBo snippet invocation will transform the API call to the following script:

                        <!DOCTYPE html>
                        <html>
                          <head>
                              :
                            <script type="text/javascript" charset="utf-8" src="/classpath/fobo/jquery.dataTables.js"></script>                          
                            <script type="text/javascript">
                            // <![CDATA[
                              $(document).ready(function() {
                                oTable = $('#example').dataTable({'bJQueryUI':true,'sPaginationType':'full_numbers'});
                              } );;
                            // ]]>
                            </script>
                              :
